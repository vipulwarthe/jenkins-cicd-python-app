Install docker, jenkins, activate python environment and clone the repo.

sudo apt update

install docker

install jenkins

activate python env

check locally

pip install -r requirements.txt

python app.py

access the app  public ip:5000

create the jenkins cicd pipeline

add docker and git credentials 

create job

and run the pipeline
